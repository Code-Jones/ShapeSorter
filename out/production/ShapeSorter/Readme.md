# Shape Sorter

This is a shapeSorter that takes in .txt files and sorts by their different shape characteristics. There are 6 different sorting algorithms with 3 different shape characteristics that can be combined with that. 

## Usage
Put the given files into the same directory as the jar and use the given jar arguments. Examples below. 

```Batch
java -jar shapeSorter.jar -h

Examples:

java -jar sort.jar –ta –sQ –fpolyfor3.txt
java -jar sort.jar –ta –sQ –fpolyfor1.txt
java -jar sort.jar –tH –Fpolyfor5.txt –sB
```
